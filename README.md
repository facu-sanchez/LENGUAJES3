# Trabajos Prácticos - Lenguajes III
Repositorio con los desarrollos y trabajos prácticos de la materia Lenguajes III.

##  Proyectos y Dependencias

### TP1:
* **Archivo:** `TP1.html`
* **Descripción:** Maquetación de una página web con diseño responsivo y menú hamburguesa funcional.
* **Tecnologías:** HTML5, CSS3 y JavaScript puro.
* **Dependencias:** Ninguna.

### TP2: Carrusel de Imágenes
* **Archivo:** `TP2.html`
* **Descripción:** Implementación de un carrusel interactivo de imágenes.
* **Dependencias requeridas:**
  * **Bootstrap 5.3.0** (Vía CDN - Framework CSS y JS para la funcionalidad del carrusel).

### TP5: Registro UCASAL
* **Archivo:** `TP5.html`
* **Descripción:** Formulario de registro de usuario con validaciones dinámicas mediante expresiones regulares (nombre, email institucional, cálculo de edad y coincidencia de contraseñas).
* **Tecnologías:** HTML5, CSS3 y JavaScript puro.
* **Dependencias:** Ninguna.

### tp7: Buscador de Personajes (Rick and Morty API)
* **Archivos:** `index.html`, `styles.css`, `script.js`
* **Descripción:** Aplicación web interactiva que consume una API externa para buscar y mostrar tarjetas de personajes por nombre o ID.
* **Dependencias requeridas:**
  * **Bootstrap 5.3.0** (Vía CDN - Para estilos y sistema de grilla).
  * **jQuery 3.7.1** (Vía CDN - Para llamadas AJAX y eventos).
  * **Google Fonts** (Tipografías *Bangers* y *Roboto*).
  * **Rick and Morty API** (`https://rickandmortyapi.com/`).
